1. Test Loading Spinner pada Submit Comment
Test untuk memastikan loading spinner muncul saat menyimpan comment dan tombol disabled selama proses.
  1. Buka comment modal untuk task tertentu
  2. Tulis komentar atau upload file
  3. Klik tombol "Kirim"
  4. Perhatikan perubahan pada tombol
Expected Result:
  • Tombol berubah menjadi "Menyimpan..." dengan spinner
  • Tombol disabled selama proses submit
  • Spinner berputar sampai proses selesai
  • Tombol kembali normal setelah selesai
2. Test Auto-Save Draft Feature
Test untuk memastikan draft comment tersimpan otomatis saat user mengetik.
  1. Buka comment modal
  2. Tulis beberapa kata di textarea
  3. Tutup modal tanpa submit
  4. Buka kembali comment modal untuk task yang sama
  5. Periksa apakah draft tersimpan
Expected Result:
  • Draft tersimpan otomatis saat mengetik
  • Indikator "Draft tersimpan otomatis" muncul
  • Draft dapat di-load kembali setelah modal dibuka ulang
  • Draft expired setelah 1 jam
3. Test Keyboard Shortcuts
Test untuk memastikan keyboard shortcuts berfungsi dengan baik.
  1. Buka comment modal
  2. Tulis komentar
  3. Tekan Ctrl+Enter (atau Cmd+Enter di Mac)
  4. Tekan Esc untuk tutup modal
Expected Result:
  • Ctrl+Enter berfungsi untuk submit comment
  • Esc berfungsi untuk tutup modal
  • Info keyboard shortcuts ditampilkan di bawah textarea
  • Placeholder text menunjukkan shortcut yang tersedia
4. Test Enhanced Error Handling
Test untuk memastikan error handling yang lebih baik dan user-friendly.
  1. Test dengan task ID yang invalid
  2. Test dengan file yang terlalu besar
  3. Test dengan network error
  4. Test dengan server error
Expected Result:
  • Error message yang spesifik dan informatif
  • User tidak bisa submit dengan task ID invalid
  • File size validation yang jelas
  • Network error handling yang graceful
5. Test Loading States
Test untuk memastikan semua loading states berfungsi dengan baik.
  1. Buka comment modal
  2. Perhatikan loading state saat fetch comments
  3. Submit comment dan perhatikan loading state
  4. Test dengan network yang lambat
Expected Result:
  • Loading spinner muncul saat fetch comments
  • Loading spinner muncul saat submit comment
  • UI disabled selama loading
  • Loading state yang smooth dan tidak flicker
6. Test Comment Validation
Test untuk memastikan validasi comment berfungsi dengan baik.
  1. Submit comment kosong tanpa file
  2. Submit comment kosong dengan file
  3. Submit comment dengan text saja
  4. Submit comment dengan file saja
Expected Result:
  • Comment kosong tanpa file tidak bisa di-submit
  • Confirmation dialog muncul untuk comment kosong dengan file
  • Comment dengan text saja bisa di-submit
  • Comment dengan file saja bisa di-submit setelah konfirmasi
7. Test UI/UX Improvements
Test untuk memastikan semua improvement UI/UX berfungsi dengan baik.
  1. Test focus states pada textarea
  2. Test hover effects pada tombol
  3. Test responsive design
  4. Test accessibility features
Expected Result:
  • Focus ring yang jelas pada textarea
  • Hover effects yang smooth
  • Responsive di semua ukuran layar
  • Keyboard navigation yang baik

  • Cross-browser compatibility
  • Graceful degradation pada network issues
9. Test Performance
Test untuk memastikan performa fitur comment tetap baik.
  1. Test dengan banyak comments

  3. Test dengan multiple modal opens
  4. Test memory usage
