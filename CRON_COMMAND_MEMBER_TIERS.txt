# Cron Job Command untuk Member Tier Update

## Command untuk Laravel Scheduler (WAJIB - hanya perlu 1 kali setup):
* * * * * cd /home/ymsuperadmin/public_html && php artisan schedule:run >> /dev/null 2>&1

**Catatan**: 
- Command ini akan menjalankan Laravel scheduler setiap menit
- Laravel akan otomatis menjalankan `members:update-tiers` setiap tanggal 1 setiap bulan jam 00:00
- Jika sudah ada command `schedule:run` untuk command lain, tidak perlu menambahkan lagi

## Command untuk Test Manual:
cd /home/ymsuperadmin/public_html && php artisan members:update-tiers

## Command untuk Update Member Tertentu:
cd /home/ymsuperadmin/public_html && php artisan members:update-tiers --member-id=1
